ADVISOR Help on Version and Type

For ADVISOR 3.0 and following, variables were introduced to define the version and the type of component being chosen by the user.  These coincide with the options available next to each component on the input screen.  The following definitions and lists are to aid the user in understanding what each of these choices represent.

Version:

This is the version of the model (ie. Simulink Block Diagram) for the component you are going to run.  For example, 'ic' for the fuel converter is a model of an internal combustion engine while 'fc' is the model for the fuel cell.  This example may change as versions and types are updated.

Type:

Type is a choice for the version that has been picked.  For example: if you picked 'ic' for the fuel converter, the types of components available for that model are spark ignition (si) and compression ignition (ci).
