What's New in ADVISOR

Version 2003 

  • Conventional drivetrain is now modeled for 4wd (2wd to 4wd with a
    change in a boolean)
  • Simplified drive cycle analysis tool
  • New fuel cell system models
  • Saber co-simulation enhancements
  •     Saber co-simulation series configuration
  •     Saber co-simulation parallel configuration
  •     Saber co-simulation battery model that allows variations between cells.
  • Adaptive control strategy for fuel cell vehicles
  • Dynamic comparison of multiple simulation results
  • More vehicle data files available from the vehicle component pull down menu
  • Design of Experiment (DOE) capability accessible from the options menu on the Simulation Setup Screen
  • Reorganization of library components for easier model building.
